# ReBest · Web oficial

Landing page estática de **[ReBest](https://agenciarebest.cl)** — agencia de publicidad y marketing digital que mezcla branding con resultados comerciales medibles.

> **Sentirás el cambio.**

## Stack

HTML + CSS + JS vanilla. Sin build step. Se despliega como sitio estático.

- `index.html` — página completa
- `assets/` — logos, serpiente (símbolo de marca), piezas de portafolio

## Desarrollo local

Basta con abrir `index.html` en el navegador, o servirlo con cualquier servidor estático:

```bash
npx serve .
# o
python -m http.server 8080
```

## Analíticas

El `<head>` incluye bloques comentados para **Google Analytics 4** y **Meta Pixel**. Descoméntalos y reemplaza los IDs de ejemplo por los reales antes de publicar.

## Despliegue

Optimizado para [Vercel](https://vercel.com) como sitio estático — conecta el repo y publica sin configuración adicional.
